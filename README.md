# Exploratory Data Analysis (EDA) using SQL and Python

## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) using **MySQL** on a data warehouse.

The objective is to analyse sales data and answer key business questions using SQL-only analysis.

---

## Tools Used
- MySQL (Workbench)
- SQL (CTEs, JOins, Aggregations, Window Functions)
- Jupyter Notebook (for CSV ingestion)

---

### 1. Database Exploration
- Explored schema, tables, and columns
- Understood data warehouse structure

### 2. Dimension Exploration
- Customer geography analysis
- Product category exploration

### 3. Date Exploration
- First and last order dates
- Sales coverage period (years of data)

### 5. Magnitude Analysis
- Revenue by category
- Customers by country and gender

### 6. Ranking Analysis
- Top & bottom products by revenue
- Top 10 customers by revenue
- Customers with the fewest orders
